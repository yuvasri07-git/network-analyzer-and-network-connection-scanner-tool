Network Analyzer Tool

How to Use:
1. Run the application as Administrator
2. Select network interface
3. Click Start Capture
4. Click Stop → Analyze packets
5. Use filters/search for analysis
6. Go to Device Scanner tab to scan network

Features:
- Packet capture and analysis
- Protocol filtering and search
- Suspicious packet detection
- Device scanner
- Export report (CSV)

Requirements:
- Windows OS
- Npcap installed
